## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. **Clone the repository**
2. **Navigate to the project directory**
3. **Install dependencies:**
  Copy code
  npm install
  yarn

5. **Run:**
  npm run dev

### Enter the backend folder
run the backend server:
  node app.js
